Welcome to sxcu's documentation!
================================

SCXU is an API wrapper for api located at `sxcu.net <https://sxcu.net/api/docs/index>`_. Pretty
much this documentation covers with some example and API reference. Currently the documentation
is in building stage.

Installation
************

SXCU can be installed using ``pip``

.. code-block:: powershell

    pip install sxcu
